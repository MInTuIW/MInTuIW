🎨 动效演示

以下效果都可在 GitHub README 中实现

1. 🌈 渐变流动

文字填充色随时间循环变化，像光在流动

2. 📊 动态统计卡片

数字轻柔呼吸闪烁，类似 GitHub Stats 卡片效果

3. 📈 进度条动画

🐍 Python80%
💻 C#65%
⚡ Vue45%

加载时逐条从 0 展开到目标值，可做成技术栈熟练度

4. 🌊 波浪分隔线

波浪线左右微微摆动，适合做区块分割

5. 🌧️ 代码雨

字符像黑客帝国一样往下掉，但要注意别太花哨抢了正文的风头

💡 用浏览器打开此文件可看到完整动画效果。GitHub README 中可嵌入 SVG 实现类似效果。
